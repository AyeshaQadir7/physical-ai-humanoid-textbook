# Code Examples Repository

This directory contains complete, runnable code examples for all four modules of the course.

## Structure

- `module-1-ros2/` - ROS 2 examples (talker, listener, services, packages)
- `module-2-simulation/` - Gazebo examples (URDF, world files, simulation control)
- `module-3-isaac/` - Isaac Sim examples (SLAM, navigation, perception)
- `module-4-vla/` - VLA integration examples (voice, LLM planning, end-to-end)

## Setup

Each module has its own `README.md` with setup instructions.

## Quick Start

```bash
# Clone this repo
git clone https://github.com/[repo]/code-examples.git

# Install ROS 2 Humble (if not already installed)
# Follow: https://docs.ros.org/en/humble/Installation.html

# Try Module 1 example
cd module-1-ros2/talker-listener
./run.sh
```

## Tested On

- Ubuntu 22.04 LTS
- ROS 2 Humble
- Python 3.10+
