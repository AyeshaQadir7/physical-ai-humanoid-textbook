# Physical AI Robotics Textbook

> **Hackathon Project**: A digital textbook for Physical AI and Robotics with a RAG-powered conversational interface.

## Overview

**Physical AI Robotics Textbook** is a full-stack project that combines a modern Docusaurus-based textbook frontend with an intelligent backend powered by OpenAI Agents SDK. Users can browse the comprehensive textbook content and ask natural language questions to receive semantically grounded answers sourced from the textbook.

### Key Features

- **Interactive Textbook**: Beautifully formatted Docusaurus site with comprehensive robotics content
- **RAG Agent API**: Retrieval-Augmented Generation using OpenAI Agents SDK for intelligent querying
- **Semantic Search**: Vector similarity search via Qdrant with Cohere embeddings
- **High-Quality Responses**: Grounded in textbook content—no hallucinations
- **Production-Ready**: Dockerized deployment, CORS support, comprehensive error handling

## Quick Start

### Prerequisites

- Python 3.10+ (backend)
- Node.js 18+ (frontend)
- API Keys:
  - OpenAI API key
  - Qdrant Cloud credentials
  - Cohere API key

### Backend Setup

```bash
cd backend

# Create and activate virtual environment
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your API keys

# Run the API
python -m uvicorn agent:app --reload --host 0.0.0.0 --port 8000
```

The API will be available at **`http://localhost:8000`** with interactive docs at **`http://localhost:8000/docs`**.

### Frontend Setup

```bash
cd frontend

# Install dependencies
npm install

# Start development server
npm run start
```

The frontend will be available at **`http://localhost:3000`**.

### Test the RAG API

```bash
curl -X POST http://localhost:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"query": "What is ROS2?"}'
```

## Architecture

### Backend: RAG Agent API

The backend implements a Retrieval-Augmented Generation (RAG) system:

```
User Query
    ↓
[FastAPI Endpoint]
    ↓
[OpenAI Agents SDK]
    ↓
[Semantic Search] → [Qdrant Vector DB]
    ↓
[Response Generation] (Grounded in retrieved chunks)
    ↓
JSON Response
```

**Components:**

1. **agent.py**: FastAPI application with `/chat` endpoint
   - Request validation using Pydantic
   - OpenAI Agents SDK integration
   - CORS middleware for cross-origin requests
   - Structured error handling

2. **retrieve.py**: Qdrant semantic search client
   - Query embedding via Cohere
   - Vector similarity search
   - Metadata retrieval for source citations

3. **ingestion/**: Content processing pipeline
   - Web crawling of Docusaurus site
   - Intelligent text chunking (512 tokens, 50-token overlap)
   - Embedding generation with batching
   - Checkpoint-based resume capability
   - Qdrant collection management

### Frontend: Docusaurus Textbook

The frontend is a static site generated by Docusaurus:

- **Responsive Design**: Mobile-friendly layout
- **Full-text Search**: Built-in search functionality
- **Code Highlighting**: Syntax highlighting for robotics code examples
- **Versioning**: Support for multiple textbook versions
- **SEO Optimized**: Automatic sitemap and meta tags
